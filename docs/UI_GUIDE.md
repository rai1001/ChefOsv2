# UI Guide (Stitch dark premium)

## Fuente de verdad
- Referencias: `docs/ui-reference/stitch/**/screen.png`
- Si existe `code.html`, se usa como base.

## Principios
- Dark mode por defecto
- Cards con bordes suaves, sombras sutiles
- Jerarquía clara: título -> filtros -> tabla/lista -> acciones

## Componentes base
- Button, Input, Select, Card, Table, Badge, Dialog, Toast

## Responsive
- PC: densidad alta (tablas)
- Tablet: navegación simple + cards/tablas compactas
- Móvil: flujos cortos, acciones grandes, listas
