
# Mapeo ChefOS -> Stitch

ChefOS Pantalla | Patrón Stitch recomendado
---|---
Login | Login screen (dark)
Dashboard | Dashboard cards + table
Listados (CRUD) | Table + filters + toolbar
Detalle (CRUD) | Form layout + side panel/dialog
Mobile tareas | Card list + primary CTA

Regla:
- Si hay varios ejemplos, elegir el más cercano a “dark premium” y reutilizar tokens.
