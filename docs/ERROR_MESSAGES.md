
# Mensajes estándar (UI)

## Auth
- AUTH_INVALID: Credenciales inválidas.
- AUTH_NO_ROLE: Tu cuenta no tiene rol asignado para este hotel.

## Validación
- VALIDATION_REQUIRED: Campo obligatorio.
- VALIDATION_INVALID: Valor inválido.

## Permisos
- PERMISSION_DENIED: No tienes permisos para realizar esta acción.

## Operaciones
- SAVE_OK: Guardado correctamente.
- SAVE_FAIL: No se pudo guardar. Inténtalo de nuevo.
- LOAD_FAIL: No se pudo cargar la información.
